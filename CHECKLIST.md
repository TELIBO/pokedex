## Emergency Questions (If You Get Stuck)

**Q: "Explain the Virtual DOM"**
A: "It's a lightweight copy of the actual DOM. React updates the Virtual DOM first, compares it with the real DOM, and only updates what changed. This makes React fast."

**Q: "What's the difference between state and props?"**
A: "State is data managed within a component and can change. Props are data passed from parent to child and are read-only in the child."

**Q: "Why use React over vanilla JavaScript?"**
A: "React provides component reusability, easier state management, better performance with Virtual DOM, and a large ecosystem of tools and libraries."

**Q: "What are hooks?"**
A: "Functions that let you use React features in functional components. useState manages state, useEffect handles side effects like API calls."
